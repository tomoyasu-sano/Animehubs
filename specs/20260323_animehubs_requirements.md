# 要件サマリー: AnimeHubs 予約システム

## アプリ概要
スウェーデン Uppsala市でアニメフィギュアを輸入販売する2人ビジネス向けの予約システム。
ユーザーが商品をカートに入れ、販売者と直接会って購入する「対面受け渡し型EC」をWebで実現する。
ログイン・オンライン決済なし。予約と連絡のみを行うシンプル設計。
ターゲットは地元スウェーデン人（30〜40代男性のアニメファン）で、英語/スウェーデン語の2言語対応。

## 技術スタック
- フロントエンド: Next.js 15 (App Router) + TypeScript
- スタイル: Tailwind CSS v4 + shadcn/ui（白黒モノトーン）
- DB / ストレージ: SQLite
- メール送信: Resend
- 多言語: next-intl（en / sv）
- パッケージマネージャ: npm
- デプロイ: ローカル（将来Vercelデプロイ予定、SEO対応必要）

## 機能要件

### 1. 商品管理
- 商品情報: 商品名、画像（最大5枚）、価格（SEK）、説明、在庫数、カテゴリ、コンディション等
- 通貨: SEK（スウェーデンクローナ）

### 2. 予約フロー
- カートに入れる → 名前・メールアドレス入力、受け渡し場所と時間帯を選択 → 予約確定 → 確認メール送信 → 対面受け渡し
- もう一つの選択肢: Instagramで友達になってDMで場所・時間を調整
- 予約キャンセル機能: 実装しない
- 受け渡し場所:
  - Uppsala Central Station（ウプサラ中央駅）
  - Stora Torget（中央駅から徒歩2-3分の広場）
  - Forumgallerian（ショッピングモール、駅近く）
- 時間帯:
  - 平日夕方 (17:00-19:00)
  - 週末午前 (10:00-12:00)
  - 週末午後 (13:00-16:00)
  （後で変更可能な設計にする）

### 3. 管理画面（/admin）
- 個別アカウント（2人分）
- シンプルなパスワード認証
- 機能: 商品CRUD、予約一覧、予約ステータス管理、売上集計

### 4. メール通知
- 予約確認時のみ（顧客へ送信）
- Resend使用

### 5. 検索・フィルタリング
- 商品検索機能あり
- カテゴリ別フィルタリング

### 6. お気に入り
- ローカルストレージベースのお気に入り機能

### 7. SEO
- メタタグ、OGP等のSEO対応あり

### 8. デザイン
コンセプト: 「モノトーン + アクセントカラー」ミニマルモダン
- ヘッダー: 黒背景、白ロゴ、ナビ（商品一覧 / カート / 言語切替）
- ヒーロー: フルワイド画像 or ダークグラデーション背景にキャッチコピー
- 商品一覧: カードグリッド（3列）、ホバーでスケールアニメーション。画像が主役
- 商品詳細: 左に画像ギャラリー（サムネイル付き）、右に情報+カートボタン
- カート: スライドインサイドパネル（右から出る）
- フッター: 黒背景、Instagramリンク、連絡先
- フォント: モダンなサンセリフ（Inter等）
- 落ち着いたコレクター向けの雰囲気、商品画像を大きく見せるクリーンなデザイン
- アニメ風装飾は控えめ
