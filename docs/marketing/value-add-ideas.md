# 付加価値アイディア集

作成日: 2026-04-03

---

## 事業目的（確定）

> **「北欧でアニメグッズといえばAnimeHubs」というポジションを取り、利益を出し続ける仕組みを作る」**

- 儲ける＝アニメが広まる。慈善ではなくビジネスとして回すことが文化普及そのもの
- まずUppsalaから。日本のアニメのHub（結節点）になる
- 安売りでは勝負しない。「日本から届く本物」のブランド価値で売る

### 儲けの構造

```
日本の中古グッズ（仕入れ安い・友人が現地調達）
  → 北欧では希少価値（売値高い）
  → 利益率が構造的に高い
  → 安売り不要
```

### ターゲット

- マニアだけじゃない。部屋に飾りたい北欧の人（スウェーデンは飾る文化あり）
- 気軽に買える感じ。敷居を低く

### 差別化ポイント

- 日本人がやるきめ細やかさ（商品の扱い、説明、梱包）
- ITを活かした体験（AR、自動化）— エンジニアだからこそ作れる
- 実物確認ルート（対面で確認できる独自機能）

---

## 前提条件・制約

- **初期投資は最小限** — お金をかけられるフェーズではない
- **副業** — 大きくなれば本業にする可能性あり
- **エンジニア** — 技術的な実装は自分でできる（＝開発コストはほぼゼロ）
- **日本の仕入れは友人が担当** — 現地パートナーあり
- **拠点はスウェーデン（Uppsala）** — 配送は現状スウェーデン国内のみ

## ロードマップ

### Phase A: 売上を作る（今〜半年）★今ここ

> **まず目の前のものを売ることに集中**

| 優先度 | 施策 | 理由 |
|--------|------|------|
| ★★★ | 商品数を増やす | 買うものがなければ始まらない。仕入れラインを太くする |
| ★★★ | Instagram/TikTok運用 | 北欧のアニメ層はSNSにいる。商品写真＋北欧の部屋に飾った写真。コストゼロ |
| ★★☆ | 紹介プログラム | 既存客が新規を連れてくる。クーポン発行で自動化 |

| 優先度 | 施策 | 理由 |
|--------|------|------|
| ★★★ | ポイント/ロイヤリティ | 2回目の購入ハードルを下げる |

| ★★☆ | 配送エリア拡大（北欧全域） | 市場を4倍に |

### Phase C: 差別化を固める（1年〜）

| 優先度 | 施策 | 理由 |
|--------|------|------|
| ★★☆ | AR「部屋に置いてみる」 | エンジニアだから作れる武器。他店にない |
| ★☆☆ | くじ引き/ミステリーBOX | エンタメ性で客単価UP。在庫処分にも |
| ★☆☆ | コミュニティ（Discord等） | ファンが集まれば勝手に広まる |

### やらないこと（今は）

- サブスクBOX → 運用重い。副業で毎月は無理
- 店舗 → 固定費が高すぎる
- 安売り → 構造的に不要。やったら終わり
- 多機能化 → まず売る。機能は後
