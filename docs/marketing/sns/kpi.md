# SNS KPI・数値目標

作成日: 2026-04-04

---

## 基本方針

**フォロワー数より「購入に繋がる指標」を追う。**

フォロワー10,000人で売上ゼロより、フォロワー300人で毎月買ってくれる人が10人いる方がいい。ニッチ市場（アニメグッズ×北欧）では数より質。

---

## 指標の優先順位

```
最重要:  サイトクリック → SNS経由の売上（実際に金になっているか）
重要:    保存数、DM問い合わせ数（購入検討のシグナル）
参考:    エンゲージメント率、フォロワー増加率
見なくていい: インプレッション数（虚栄の指標）
```

---

## フェーズ別KPI

### Month 1-3: 種まき期

| 指標 | 目標 | 補足 |
|------|------|------|
| フォロワー数 | 200-500 | ニッチ市場では十分なスタート |
| 投稿数 | 週4-5回（フィード+ストーリーズ計） | クオリティを落とさない範囲で |
| エンゲージメント率 | 5-10% | いいね+コメント÷フォロワー数。3%切ったら内容を見直す |
| プロフィールアクセス | 週50+ | 「気になってプロフィール見に来た」数 |
| サイトクリック | 週10+ | プロフィールのリンクからサイトへの流入 |

**この時期の本当のゴール**: 投稿の型を見つけること。何が伸びて何が伸びないかのデータを溜める。

### Month 4-6: 成長期

| 指標 | 目標 | 補足 |
|------|------|------|
| フォロワー数 | 500-1,500 | Reelsがハマれば一気に伸びる可能性あり |
| エンゲージメント率 | 4-8% | フォロワー増加に伴い率は下がるのが自然 |
| サイトクリック | 週30+ | |
| SNS経由の売上 | 月3-5件 | 「SNSから買う人がいる」を確認する段階 |
| 保存数 | 投稿あたり5+ | 「後で見返したい」= 購入検討中のシグナル。いいねより重要 |

### Month 7-12: 収穫期

| 指標 | 目標 | 補足 |
|------|------|------|
| フォロワー数 | 1,500-3,000 | ニッチ市場で3,000あれば「スウェーデンのアニメ垢」として認知される |
| SNS経由の売上 | 月10-20件 | |
| UGC投稿 | 月3+ | 購入者が自発的に投稿してくれる数 |
| DM問い合わせ | 週3+ | 「これまだありますか？」= 需要のシグナル |

---

## 投稿頻度ガイドライン

| 形式 | 推奨頻度 | 注意 |
|------|---------|------|
| フィード投稿 | 週3-4回が上限 | 多すぎるとフォロワーのTLを埋めて逆効果 |
| ストーリーズ | 毎日OK | 24hで消えるので気軽に。ここを多くする |
| Reels/TikTok | 週2-3回 | 伸びるパターンを見つけたら類似を量産 |

「量を増やす」より「1投稿の質を少し上げる」方がリターンが大きい。

## 計測方法

### Instagramインサイト（無料）

クリエイターアカウントに切り替えると利用可能（切り替えは無料）。

| 指標 | 確認場所 | 備考 |
|------|---------|------|
| サイトクリック | プロフィール → インサイト →「外部リンクのタップ」 | プロフィールリンク経由の流入数 |
| プロフィールアクセス | インサイト → 概要 | 投稿から興味を持った人の数 |
| エンゲージメント | 各投稿 → インサイト | いいね・コメント・保存・シェアの合計。率は自分で計算 |
| 保存数 | 各投稿 → インサイト | 投稿ごとに確認可能 |
| リーチ | インサイト → 概要 | 何人に届いたか |
| フォロワー属性 | インサイト → フォロワー | 年齢・性別・所在地・アクティブ時間帯 |
| ストーリーズ反応 | ストーリーズ → インサイト | タップ数、返信、離脱ポイント |

### Instagramだけでは追えない指標

| 指標 | 計測方法 |
|------|---------|
| SNS経由の売上 | Stripeダッシュボードの売上推移と、Instagramの投稿タイミングを突き合わせて傾向を掴む |
| DM問い合わせ数 | 手動カウント（週1でスプレッドシートに記録する程度でOK） |
| UGC投稿数 | `#animehubs` のハッシュタグ検索で確認 |

> 分析ツールは **Instagramインサイト＋Stripeダッシュボード** の2つだけで運用する。
> 管理画面を増やさない。厳密なトラッキングは売上規模が大きくなってから検討すればよい。

## 月次振り返りチェックリスト

月末に以下を確認し、翌月の方針を調整する:

- [ ] フォロワー増減数
- [ ] エンゲージメント率の推移
- [ ] サイトクリック数
- [ ] SNS経由の売上件数
- [ ] 最も保存された投稿（→ 類似コンテンツを増やす）
- [ ] 最もリーチした投稿（→ 何が拡散されたか分析）
- [ ] DM問い合わせの傾向（→ 仕入れの参考にする）
