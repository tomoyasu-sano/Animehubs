# フィギュア商品撮影ガイド

AnimeHubsの商品写真を統一感のあるプロクオリティで撮影するためのガイド。

---

## 対象商品

- ワンピース（ルフィ、ゾロ、ナミ等）
- 葬送のフリーレン（フリーレン、フェルン、ヒンメル等）
- ドラゴンボール（悟空、ベジータ、フリーザ等）
- その他アニメフィギュア全般

---

## 1. 撮影環境セットアップ

### 背景

- **黒背景を標準とする**（サイトのダークテーマとの統一感）
- 模造紙（黒）を曲線をつけて敷き、床と壁の境目を消す（無限背景）
- 白背景はSNS投稿用・比較用のサブカットとして使用

### 照明

| 配置 | 説明 |
|------|------|
| メインライト | 被写体の斜め45度上から。ディフューザー越しに当てる |
| フィルライト | メインの反対側から弱めに。影を和らげる |
| レフ板 | 光が届かない側に白い板を置いて反射させる |

- **直射光は厳禁**（テカリ・硬い影の原因）
- 自然光を使う場合は窓際 + レフ板（白い段ボールで代用可）
- 色温度は**やや寒色寄り（5500〜6500K）** でサイトのトーンに合わせる

### 撮影台

- 黒アクリル板を使うと鏡面反射で高級感が出る
- 透明アクリルライザーで高さを変えると構図に変化がつく

## 2. カメラ設定

スマートフォンでも十分対応可能。

| 項目 | 推奨設定 |
|------|----------|
| モード | ポートレート or マニュアル |
| ISO | できるだけ低く（100〜400） |
| ズーム | 2〜3倍（レンズ歪み軽減） |
| グリッド | ON（三分割法の補助） |
| HDR | OFF（色味がブレる） |
| フラッシュ | OFF（常にライティングで対応） |

**三脚は必須**。手ブレがプロとアマの最大の差。

## 4. 作品別撮影ポイント

### ワンピース

- **ルフィ系**: 腕を伸ばすポーズは横構図で余白を広めに
- **ゾロ系**: 刀のディテールを接写で1カット追加
- **ナミ・ロビン系**: ポーズの曲線美が映る斜め構図を重視
- 全体的にダイナミックなポーズが多いため、**アングルを低め（見上げ）** にすると迫力が出る

### 葬送のフリーレン

- **落ち着いた雰囲気**を活かす。ライティングは柔らかめ
- 杖・魔法エフェクトパーツがある場合はバックライトで透過感を演出
- フリーレンの表情は繊細なので**顔アップは必ず撮る**
- フェルン・ヒンメルとの**複数体並び**カットも訴求力が高い

### ドラゴンボール

- バトルポーズが多いため**見上げアングル + 黒背景**で迫力重視
- 気のエフェクトパーツがあればバックライトで光らせる
- 超サイヤ人の髪は上方からのライトでグラデーションを強調
- 大型フィギュア（大猿、神龍等）はスケール感が伝わる引きカットも

## 5. 撮影後の編集

### 基本補正

1. **明るさ**: やや暗めに撮影 → 編集で持ち上げる（白飛び防止）
2. **コントラスト**: 少し上げてメリハリを出す
3. **色温度**: 寒色寄りに微調整（サイトトーンとの統一）
4. **シャープネス**: 軽くかける（かけすぎるとノイズが目立つ）

### サイト掲載用リサイズ

| 用途 | サイズ | フォーマット |
|------|--------|-------------|
| 商品メイン画像 | 1200×1200px | WebP |
| サムネイル | 400×400px | WebP |
| OGP / SNS共有 | 1200×630px | PNG or WebP |

- アスペクト比は**1:1（正方形）** を基本とする
- ファイルサイズは200KB以下を目標（Cloudflare Images経由で最適化）

## 6. 機材ロードマップ

### Phase 1: スマホだけで始める（初期費用 ¥0）

スマホ + 家にあるもので十分プロっぽく撮れる。

| 必要なもの | 家にあるもので代用 |
|-----------|-------------------|
| カメラ | スマホ（ポートレートモード） |
| 三脚 | 本を積む / コップに立てかける |
| 背景 | 黒いTシャツ / 黒い画用紙 |
| レフ板 | 白いコピー用紙 / アルミホイル |
| ディフューザー | 薄手の白いレジ袋 / クッキングシート |
| 撮影台 | 黒い本の表紙 / 黒いファイル |

**スマホ撮影のコツ**
- 窓際の自然光が最強のライト（曇りの日がベスト、直射日光は避ける）
- 2〜3倍ズームで撮る（広角の歪みを防ぐ）
- タイマー撮影（2秒）でシャッター時の手ブレを防ぐ
- 机の上に肘を固定して安定させる

## 7. チェックリスト（撮影前）

- [ ] フィギュアのホコリを除去したか（メガネ拭き等で拭く）
- [ ] 背景にシワ・汚れ・余計なものが写り込んでいないか
- [ ] 光の向き・影が不自然でないか
- [ ] スマホが安定しているか（三脚 or 固定）
- [ ] グリッド表示がONか
- [ ] フラッシュがOFFか
- [ ] HDRがOFFか
- [ ] 前回の写真と色味・明るさが統一されているか
