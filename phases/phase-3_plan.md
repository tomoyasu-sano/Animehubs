# Phase 3: 管理画面

## Phase概要
管理者認証（シンプルなパスワード認証 + JWT セッション）、商品CRUD画面（画像アップロード含む）、予約管理画面、売上集計ダッシュボードを実装する。

## 実装範囲

### 1. 管理者認証
- ログインページ（/admin/login）
- POST /api/admin/auth — ログイン（bcrypt検証 + JWT発行）
- ミドルウェア — 管理画面ルートのトークン検証
- ログアウト機能
- lib/auth.ts — 認証ヘルパー関数

### 2. 管理画面レイアウト
- admin/layout.tsx — サイドバーナビゲーション
- ダッシュボードページ（/admin）— 概要表示

### 3. 商品管理
- 商品一覧（/admin/products）— テーブル表示、検索
- 商品新規作成（/admin/products/new）— フォーム + 画像アップロード
- 商品編集（/admin/products/[id]/edit）
- 商品削除（確認ダイアログ付き）
- POST /api/admin/products — 商品作成
- PUT /api/products/[id] — 商品更新
- DELETE /api/products/[id] — 商品削除
- POST /api/upload — 画像アップロード（sharp でリサイズ）

### 4. 予約管理
- 予約一覧（/admin/reservations）— テーブル表示、ステータスフィルタ
- 予約ステータス更新（pending → confirmed → completed）
- GET /api/reservations — 予約一覧（admin認証必須）
- PUT /api/reservations/[id] — ステータス更新

### 5. 売上集計
- 売上集計ページ（/admin/sales）
- 期間別売上（日/週/月）
- カテゴリ別売上
- ステータス別予約件数
