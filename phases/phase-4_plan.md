# Phase 4: SEO + レスポンシブ + パフォーマンス最適化

## Phase概要
SEO対応（メタタグ、OGP、サイトマップ）、レスポンシブの最終調整、パフォーマンス最適化（画像最適化、ローディングUI）を実装する。仕上げフェーズ。

## 実装範囲

### 1. SEO対応
- 各ページの metadata（title, description, openGraph）
- 動的OGP（商品詳細ページ）
- sitemap.ts（動的サイトマップ生成）
- robots.ts
- JSON-LD 構造化データ（Product, WebSite）

### 2. レスポンシブ最終調整
- モバイル（< 640px）: 1列グリッド、ハンバーガーメニュー
- タブレット（640px〜1024px）: 2列グリッド
- デスクトップ（> 1024px）: 3列グリッド
- 画像ギャラリーのモバイル対応（スワイプ）
- カートサイドパネルのモバイル対応（フルスクリーン）

### 3. パフォーマンス最適化
- Next.js Image コンポーネントの最適化
- ローディングスケルトンUI
- ページ遷移のローディング状態
- lazy loading（商品画像）
- 不要な再レンダリングの排除

### 4. アクセシビリティ
- キーボードナビゲーション
- aria-label の追加
- フォーカスリングの視認性
- スクリーンリーダー対応（alt テキスト等）
